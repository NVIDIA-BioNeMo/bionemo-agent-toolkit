# Meta-Skills

Portable composition skills that orchestrate multiple BioNeMo NIM skills into
common scientific workflows. Skill metadata uses `license: Apache-2.0 AND
CC-BY-4.0`.

Each directory contains:

- `SKILL.md`: runtime workflow instructions for the multi-NIM pipeline.
- `evals/`: packaged task eval definitions for Agent Skills publishing and
  compatibility.

Use individual NIM skills under `skills/` for API-specific payload and Docker
details.
